# Toprank

Toprank is an open-source Claude Code plugin from [nowork-studio/toprank](https://github.com/nowork-studio/toprank) for SEO, Google Ads, and content optimization workflows.

## What it provides

- Search Console-driven SEO analysis and traffic drop triage
- Google Ads audits, spend optimization, and ad copy workflows
- Keyword research, meta tag optimization, schema markup generation, and CMS setup helpers
- Repo-aware Claude Code workflows that can inspect codebases and apply SEO fixes directly

## Upstream project

- Repository: https://github.com/nowork-studio/toprank
